What is inheritance and why is it important?

Inheritance is a fundamental concept in object-oriented programming that allows a new class to inherit properties and behaviors from an existing class. This principle promotes code reusability and a establishes a natural hierarchy between classes, which can reduce redundancy and improve maintainability.

One major benefit of inheritance is that it allows for polymorphism. This means that, a simgle function can operate on different classes through a common interface, enabling more flexible and dynamic code. For example, if you have a base class called Shape, you can create subclasses like Circle and Rectangle, each with its own methods to calculate area while still being able to use a common method to display the shape.

Which provides common mthods for all derived classes like BreathingActivity, listingActivity, and ReflectionActivity. This structure promotes code reusability and allows for easy expansion.
